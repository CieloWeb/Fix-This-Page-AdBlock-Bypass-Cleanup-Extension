# Fix This Page – AdBlock Bypass & Cleanup Extension 🚀

“Fix This Page” is a powerful, privacy-first browser extension for Chrome and Firefox that removes annoying overlays, bypasses adblock detection, enables dark mode, and boosts readability – all with a single click.

---

## 🛠 Features

- 🧼 One-click cleanup of overlays, popups, and ads
- 🕶️ Dark Mode for any website
- 📖 Text boosting for easier reading
- 🚫 Stops animations and distractions
- 🧠 Privacy-first: no data collected, ever
